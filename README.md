# FFI Request Collection

> The place to report bugs, request features, and share feedback for the **my-taxes** app.
> No account required to browse — just open an issue to get started.

---

## What can you do here?

Pick the template that best matches your situation. Each one asks the right questions to help us act on it quickly.

| Template | Use it when... |
|---|---|
| 🐛 **Bug report** | Something isn't working as expected |
| 🚀 **Feature request** | You have an idea for something new or an improvement |
| 💬 **Feedback** | General impressions, things you love, or things that confused you |

👉 [**Open an issue**](https://github.com/noxs-jj/ffi-request-collection/issues/new/choose)

---

## How it works

1. **Pick a template** — click the link above and choose the one that fits. Blank issues are disabled so nothing gets lost.
2. **Fill in the form** — each template asks specific questions. The more detail you give, the faster we can act on it.
3. **Submit** — your issue is now visible to the team. Feature requests and bugs that gain traction get moved into the main development backlog at [noxs-jj/my-taxes](https://github.com/noxs-jj/my-taxes).

> This repo is for collecting input only — no code lives here.
> The app source code and development issues are tracked at **github.com/noxs-jj/my-taxes**.

---

## Guidelines

- **Search before opening** — someone may have already reported the same thing. Add a reaction or comment to an existing issue instead of opening a duplicate.
- **Be specific** — vague reports like "it doesn't work" are hard to act on. Steps to reproduce, screenshots, and your browser version go a long way.
- **Be kind** — this is a small project built in the open. Constructive feedback is always welcome.
